## Installation

1. Prerequisites

[nodejs](https://nodejs.org)
[yarn](https://yarnpkg.com)

2. Clone the project

`git clone https://github.com/<user>/<project>`

3. Initialize the app

`npm run install` (for nodejs)

or

`yarn` (for yarn)

4. Start the app

`npm run start` (for nodejs)

or

`yarn start` (for yarn)
